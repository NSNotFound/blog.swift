## Requirements

1. PostgreSQL
2. Ubuntu 14/15(not tested) or Mac OS X 10.11
3. Swift 2.2
4. watchdog
5. Homebrew for OS X



## Usage

1. Install Swift 2.2
2. `make bootstrap`
3. `make initdb`
4. `make build`
5. `make start`
6. Open http://localhost:4000
7. Press `Ctrl+c` to stop the server
8. `make watch` and change any Swift code...
